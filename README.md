# poebat.today - based IT courses for beginners/rookies

[poebat.today](https://poebat.today)

## Setup

```bash
bun install
```

## Development Server

Start the development server on `http://localhost:3000` y'know:

```bash
bun run dev
```

## Production

Build the app:

```bash
bun run generate
```

Locally preview production build (but we don't care):

```bash
bun run preview
```
